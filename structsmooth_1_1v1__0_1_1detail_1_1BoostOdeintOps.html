Detailed Description

boost::odeint Stepper operations for Manifold types.

boost::odeint Butcher tableaus are evaluated by weighted calculations of the form y = Σ_{i=1}^n alpha_i x_i which are generically implemented.

However, for the special case of butcher tableaus it holds that alpha_1 = 1, and furthermore x_1 is always of the state type while x_2 ... x_n are of the derivative type. The scale sum can therefore be generalized to the Lie group case as

y = x_1 * exp(Σ_{i=2}^n alpha_i x_i)

The methods below inject those calculations into boost:odeint to enable numerical integration on Lie groups. For succintness we implement a single method using variadic templates.
